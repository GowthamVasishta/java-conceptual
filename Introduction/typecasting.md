# Java basics

## Type casting
Converting one primitive type to another primitive type, one reference type to another reference type is called **casting**.

Implicit     | Explicit
-------------|----------
`int n = 100;` | `long l = 100;`
`long l = n;`  | `int n = l; //error`
`//implicit conversion`| `int n = (int) l;`
